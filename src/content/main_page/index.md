---
title: HAKU
subtitle: "A simple Astro theme"
---

HAKU 是一个基于 Astro 框架的静态博客主题，它专注于以极简的风格展示博客以及提升读者的阅读体验。HAKU 可以展示个人项目和笔记，同时，它也为喜爱记录生活的人提供了一个很好的平台。

HAKU 使用 Tailwind CSS 作为 CSS 引擎，并基于此设计了诸多响应式设计，支持明暗主题的切换和基于 Pagefind 的站内搜索。HAKU 响应迅速，风格干净。

由 [@Karlbaey](//github.com/Karlbaey/astro-theme-haku) 设计，基于 [Astro](//astro.build) 前端框架。
